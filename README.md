🚀 Ardeva POS - Modern Cashier System

Ardeva POS adalah solusi aplikasi kasir berbasis mobile yang dirancang untuk kecepatan, kemudahan penggunaan, dan manajemen stok yang akurat. Dibangun menggunakan Flutter untuk pengalaman UI yang mulus dan Laravel API sebagai tulang punggung data yang kokoh.

✨ Fitur Utama
⚡ Transaksi Cepat: Input pesanan hanya dengan beberapa ketukan.

📦 Manajemen Inventaris: Pantau stok secara real-time dengan notifikasi stok menipis.

📊 Laporan Penjualan: Dashboard ringkasan pendapatan harian, mingguan, dan bulanan.

🖨️ Cetak Struk: Dukungan cetak struk via Bluetooth Thermal Printer.

👥 Multi-User: Level akses berbeda untuk Admin dan Kasir.

☁️ Sinkronisasi Cloud: Data tersimpan aman dan sinkron antar perangkat.

🛠️ Stack TeknologiKomponenTeknologi
FrontendFlutter (Dart)
BackendLaravel (PHP)
Database Supabase 
ManagementBloc / Provider / GetXAPIRESTful API

🚀 Cara Instalasi

Prasyarat
Flutter SDK terpasang.

Laravel Backend sudah berjalan (API URL siap).

Langkah-langkah
Clone Repository

Bash
git clone https://github.com/ariffatur22/Flutter-pos-android.git
Masuk ke Direktori

Bash
cd Flutter-pos-android
Install Dependencies

Bash
flutter pub get
Konfigurasi API
Buka file lib/config/constants.dart (atau lokasi config Anda) dan sesuaikan baseUrl dengan alamat Laravel API Anda.

Jalankan Aplikasi

Bash
flutter run
